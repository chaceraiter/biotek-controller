# Summary: BioTek Gen5 Getting Started Guide

- Overview of Gen5 as integrated data collection and analysis software for BioTek multimode and single-mode readers.
- Describes core capabilities: endpoint, kinetic, spectral scanning, and well-area scanning, plus imaging workflows (Augmented Microscopy).
- Includes installation guidance and computer system recommendations for imaging vs non-imaging use.
- Provides compatibility and software level comparison information (instrument compatibility, imaging capabilities).
- Positions the document as a high-level onboarding guide rather than a protocol or device driver reference.
